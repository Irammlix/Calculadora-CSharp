# Calculadora en C# / Windows Forms

![.NET](https://img.shields.io/badge/.NET-Framework_4.8-purple)

Proyecto de práctica desarrollado en C# y Windows Forms. Implementa una calculadora funcional con historial y repetición de operaciones.

## Funcionalidades

- Suma, resta, multiplicación, división y módulo.
- Manejo de errores (división entre cero).
- Historial de operaciones.
- Repetición de operaciones mediante "=".

## Cómo probarlo

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Irammlix/Calculadora-CSharp.git
   ```

2. Abre el archivo `.sln` en Visual Studio.

3. Ejecuta el proyecto.

## Estructura

- `FormCalculadora.cs` → Lógica principal de la calculadora.
- `Program.cs` → Punto de entrada de la aplicación.
